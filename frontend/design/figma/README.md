# Figma: Galeria Estática (5 imagens)

Este SVG foi criado para importação direta no Figma e representa um bloco de galeria com 5 imagens estáticas, seguindo a proporção mostrada no screenshot.

## Como usar no Figma
1. Abra o Figma e crie/abra um arquivo de design.
2. Arraste o arquivo `static-5-images.svg` para a área de trabalho do Figma, ou use File → Import.
3. Cada cartão (retângulo) e texto será importado como camadas editáveis.
4. Para substituir por suas imagens:
   - Selecione um cartão (retângulo).
   - No painel de propriedades, em Fill, troque o preenchimento para Image e faça upload da imagem.
   - Ajuste `Scale`, `Fill` ou `Fit` conforme necessário.
5. Opcional: ajuste o tamanho do frame conforme seu layout.

## Dicas
- As dimensões atuais: 1219.32 x 446.18 px. Pode ampliar sem perder a proporção.
- Os cantos arredondados estão em 16px; ajuste conforme seu guideline.
- Use `Auto Layout` no Figma caso queira tornar responsivo entre versões de página.

## Imagens do projeto
Se quiser usar imagens do próprio projeto (React), você pode referenciá-las do diretório `frontend/src/assets`. No Figma, basta fazer upload manual dessas imagens:
- `frontend/src/assets/img-man.png`
- `frontend/src/assets/img-woman.png`
- `frontend/src/assets/img-tshirt.png`
- `frontend/src/assets/img-man-woman.png`
- `frontend/src/assets/img-woman-fita.png`

> Observação: o SVG não embute imagens automaticamente para manter o arquivo leve e portátil. Substitua os cartões pelas imagens diretamente no Figma.
